# Sims 4 Tuning
What happened to the tuning?

## What is this?
This repository contains all of The Sims 4's XML Tuning as produced by Scumbumbo's XML Extractor with the following settings:
* Include Strings from: **Strings_ENG_US**
* Include XML References: ✔️
* Create Reference Files (STBL and Names): ✔️
* Create XML File Index: ✔️
* Include JAZZ Resources: ✖️
* XML File Naming: *varies based on folder*
* Normalize `n="."` and `t="."` Attribute Order: **n Attribute First**

## Who is it for?
Sims 4 mod creators who want an easily accessible, technical reference of the state of the tuning in game versions released since this repository was created, and precise differences derived by Git.
